\doxysubsection{Подробное описание}
Абстрактный базовый класс, представляющий элемент библиотеки. 

Класс содержит общие поля, такие как название, автор и год публикации. Используется как основа для производных классов \doxylink{classorg_1_1example_1_1_book}{Book} и \doxylink{classorg_1_1example_1_1_magazine}{Magazine}.
